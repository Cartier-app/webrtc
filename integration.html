Quick Start

Method 1: JavaScript SDK Recommended

The easiest way to integrate CallConnect into your application.

Step 1: Include the SDK

<script src="https://cartier-app.github.io/webrtc/embed.js"></script>

‚úì Ready to use! The SDK automatically detects the correct server URL from the script source.

Step 2: Add a container

<div id="callconnect-container"></div>

Step 3: Initialize

<script>
  const callConnect = new CallConnectEmbed({
    containerId: 'callconnect-container',
    mode: 'both',
    width: '100%',
    height: '600px',
    username: 'John Doe',
    onReady: function() {
      console.log('CallConnect is ready!');
    }
  });
  
  callConnect.init();
</script>

Method 2: iFrame Embed

Simple iframe integration for quick embedding.

<iframe 
  src="https://cartier-app.github.io/webrtc/?embed=true&mode=both&username=Guest"
  width="100%"
  height="600px"
  frameborder="0"
  allow="camera; microphone; display-capture"
  style="border-radius: 12px;"
></iframe>

Note: Direct iframe embedding provides less control compared to the JavaScript SDK. We recommend using the SDK (Method 1) for full programmatic access and better integration.

Advanced Configuration

Customize every aspect of the integration.

<script>
  const callConnect = new CallConnectEmbed({
    containerId: 'callconnect-container',
    mode: 'video',
    theme: 'dark',
    width: '800px',
    height: '600px',
    position: 'modal',
    autoJoin: false,
    roomCode: 'ABC123',
    username: 'Custom User',
    showControls: true,
    
    onReady: function(data) {
      console.log('Ready:', data);
    },
    
    onJoin: function(data) {
      console.log('Joined room:', data);
    },
    
    onLeave: function(data) {
      console.log('Left room:', data);
    },
    
    onError: function(error) {
      console.error('Error:', error);
    }
  });
  
  callConnect.init();
  
  callConnect.createRoom('My Username');
  
  callConnect.show();
</script>

Configuration Options

Option Type Default Description
containerId string 'callconnect-container' HTML element ID for the container
mode string 'both' 'video', 'audio', or 'both'
width string '100%' Width of the embed
height string '600px' Height of the embed
position string 'inline' 'inline', 'floating', or 'modal'
autoJoin boolean false Auto-join room on load
roomCode string null Pre-set room code
username string 'Guest' Default username
baseUrl string Auto-detected from script URL CallConnect server URL (defaults to https://cartier-app.github.io/webrtc)

üîí Security Note

The SDK uses secure cross-origin messaging with origin validation. The iframe only accepts messages from the parent origin that embedded it, preventing unauthorized control by malicious scripts.
